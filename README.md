# Inriser

Inriser is a modern landing page built with **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**. It’s designed as a clean, responsive marketing site with a polished UI, subtle motion, and reusable components.

**Live site:** https://inriser.vercel.app

---

## What’s inside

- **App Router (Next.js)** project structure under `app/`
- Component-driven sections (hero, features, CTA, founders, contact)
- Styled with **Tailwind CSS** + `tailwind-merge`
- UI primitives from **Radix UI**
- Animations via **Framer Motion**
- Theme support using **next-themes**

---

## Tech stack

- **Framework:** Next.js
- **Language:** TypeScript
- **UI:** React, Tailwind CSS, Radix UI
- **Animation:** Framer Motion
- **Icons:** lucide-react

---

## Project structure (high level)

- `app/` – routes and pages (App Router)
- `components/` – layout + page sections
- `lib/` – shared utilities/helpers
- `public/` – static assets
- `tokens/` – design tokens (if used by the UI)

---

## Getting started

### Prerequisites

- Node.js (recommended: latest LTS)
- npm (or pnpm/yarn/bun)

### Install

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open http://localhost:3000

### Production build

```bash
npm run build
npm run start
```
